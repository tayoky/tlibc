# tlibc
tlibc is a portable standard c library originally written for the [Stanix operating system](https://github.com/tayoky/stanix) with the goal of being complete to compile program such as gcc, binutils or git.

# software
with more than 200 functions tlibc can be used with program such as 
- all Stanix's userspace programs
- [tutils](https://github.com/tayoky/tutils)
- [tash](https://github.com/tayoky/tash)
- [tcc](https://github.com/tinycc/tinycc)
- binutils (by disabling tls)
- make
